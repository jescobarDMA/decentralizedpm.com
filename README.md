# decentralizedpm.com

Marketing website for **Decentralized Property Management (DPM)** — full-service residential and commercial property management in Houston, TX.

**Tagline:** Modern Property Management, Decentralized.
**Contact:** info@decentralizedpm.com · 713-299-2339

## Stack

Plain static site — hand-written HTML + CSS + vanilla JS. No framework, no build step.

## Local preview

```bash
open index.html
```

Or serve with any static server:

```bash
python3 -m http.server 8000
```

## Structure

- `index.html` — single-page site (hero, problem, services, process, about, proof, CTA split, footer)
- `styles.css` — all styles, DPM brand tokens at `:root`
- `assets/fonts/` — Instrument Sans (headings), Lora (body)
- `assets/images/` — logo + favicon
- `assets/video/` — hero background video (placeholder — drop in MP4 when ready)
